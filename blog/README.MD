## Адаптивный шаблон для блога, 9 разных секций.

Что внутри:
* Bootstrap 3
* LESS
* jQuery 3
* Gulp 4
----
#### Установка зависимостей
    $ npm i

#### Полная сборка проекта
    $ gulp

#### Сборка стилей
    $ gulp buildStyles

#### Сборка скриптов
    $ gulp buildScripts
