===========================
Сборка на GULP Task manager
===========================

Установка зависимостей
$ npm i

Полная сборка проекта
$ gulp

Сборка стилей
$ gulp buildStyles

Сборка скриптов
$ gulp buildScripts

Копировать HTML
$ gulp copyPages

Копировать CSS-библиотеки
$ gulp copyLibCss

Копировать JS-библиотеки
$ gulp copyLibJs

Копировать шрифты
$ gulp copyFonts

Копировать картинки
$ gulp copyImages
