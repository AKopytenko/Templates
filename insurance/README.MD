## Адаптивный шаблон для страховых компаний.

Что внутри:
* Bootstrap 3
* LESS
* jQuery 3
* Gulp 4
----
#### Установка зависимостей
    $ npm i

#### Полная сборка проекта
    $ gulp

#### Сборка стилей
    $ gulp styles

#### Сборка скриптов
    $ gulp scripts
