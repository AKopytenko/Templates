Адаптивный шаблон [Gulp, LESS, Bootstrap 4*, jQuery 3*]
---
Установка зависимостей
$ npm i

Полная сборка
$ gulp

Сборка стилей
$ gulp buildStyles

Сборка скриптов
$ gulp buildScripts
